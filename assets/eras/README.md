# Era images

Place nine identically sized images in this directory:

- `era-0.png` — Colonial Boot
- `era-1.png` — Founding Build
- `era-2.png` — Continental Expansion
- `era-3.png` — Industrial Build
- `era-4.png` — Electrified Nation
- `era-5.png` — Mass Production
- `era-6.png` — Space & Silicon
- `era-7.png` — Networked America
- `era-8.png` — Present Day

Recommended source dimensions: **160 × 90 pixels**.

The UI displays them in a fixed 16:9 frame with `object-fit: contain` and
pixelated image rendering. Missing images display an `ERA MAP` placeholder.
